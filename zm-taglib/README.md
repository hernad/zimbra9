Steps to build
--------------

1. Clone the repo

  `git clone ssh://git@stash.corp.synacor.com:7999/zimbra/zm-taglib.git`

2. Run the command

    `ant clean publish-local`
 
## Dependencies
- `zm-store`
- `zm-soap`
- `zm-client`
- `zm-common`
- `ThirdParty Jars`

## Artifacts
- `zm-taglib.jar`
- Class files for `/src`
